# 会议室座位定位规则（修正版 v2.0）

> **版本说明**：本版取消双方人数均为偶数时的交叉对坐规则，统一执行"1号位必须对坐"原则。同时引入字母座位编号体系，避免与人员编号混淆。

---

## 一、两套编号体系说明

### 1.1 座位编号（字母体系）

座位编号使用**小写字母**，只是便于理解以及与人员编号区分，不需要在会议室平面图上标注，不因参会人员变化而改变。

| 区域 | 座位编号 | 物理位置说明 |
|:---|:---|:---|
| **长边A（客方）** | a、b、c、d、e、f、g、h | 从电子白板侧向短边侧依次为 a→h |
| **短边** | i、j | 从靠近客方侧向靠近主方侧，依次为 i→j |
| **长边B（主方）** | r、q、p、o、n、m、l、k | 从靠近短边侧到电子白板侧（从门1侧向门2侧）依次为 r→k |

### 1.2 人员编号（数字体系）—— 随参会人员变动

人员编号使用**阿拉伯数字**（1号、2号、3号……），指代参会人员职务高低次序，按"以左为尊"规则动态分配，每次会议根据实际参会人员重新排布。

### 1.3 两套编号的关系

- **座位编号（字母）** = 平面图中的固定位置，类似"座位地址"
- **人员编号（数字）** = 当前坐在该位置上的人，类似"住户"

## 二、情况一：主方人数 ≤ 7（无论客方几人）

| 项目 | 内容 |
|:---|:---|
| **判定条件** | 主方 ≤ 7 |
| **客方1号位** | **座位 d** |
| **主方1号位** | 与客方1号位对坐（座位 o） |
| **排布规则** | 双方均按"以左为尊"正常展开 |
| **原因** | 主方未满员，客方1号位优先定在座位 d，主方1号位与之对坐 |

---

## 三、情况二：主方满员（=8）且 客方不满员（≤7）

| 项目 | 内容 |
|:---|:---|
| **判定条件** | 主方 = 8 且 客方 ≤ 7 |
| **客方1号位** | **座位 e** |
| **主方1号位** | 与客方1号位对坐（座位 n） |
| **排布规则** | 双方均按"以左为尊"正常展开 |
| **原因** | 主方8人占满8座，以左为尊向左展开需4空位，1号位必须在座位 n；客方1号位与之对坐在座位e|

---

## 四、情况三：主方满员（=8）且 客方满员（=8）

| 项目 | 内容 |
|:---|:---|
| **判定条件** | 主方 = 8 且 客方 = 8 |
| **客方1号位** | **座位 d** |
| **主方1号位** | 与客方1号位对坐（座位 o） |
| **排布规则** | 客方按"以左为尊"正常展开；主方1-7号按"以左为尊"正常排布，8号作为列席位置于主方长边唯一空位（座位 k） |
| **原因** | 客方8人时1号必须定在座位 d，主方1号位需与客方1号对坐在座位o，导致主方1号靠电子白板侧只能排3人（2号、4号、6号），8号无处可去，故调整至剩余空位（座位 k） |

### 4.1 席位调整说明

- **主方1-7号**：按"以左为尊"正常排布
- **主方8号**：作为列席位，置于主方长边唯一空位（座位 k）

### 4.2 视觉标识

- 平面图中主方8号蓝色方块采用对比色区分样式（如#c96442），尽量避免破坏整体布局美观。

### 4.3 平面图备注

在平面图空白处添加小字备注：

> 注：双方均满员时，按以左为尊规则主方8号理论位置应在6号左侧，因席位限制调整至剩余空位作为列席。

## 五、核心原则速查

| 原则 | 含义 |
|:---|:---|
| **以左为尊** | 2号在1号的左手边，3号在右手边，依次左右交替 |
| **居中为尊** | 1号永远在中间；偶数时1号2号同时居中 |
| **面门为尊** | 客方面对门（长边A），主方背对门（长边B） |
| **1号位必须对坐** | 无论主/客方人数为奇数或偶数，1号位始终在同一垂直线上对坐 |
| **满员优先** | 客方满员（8人）时，以客方为准确定1号位 |
| **列席位标识** | 双方均满员时，主方8号以对比色底色标识为列席位 |
