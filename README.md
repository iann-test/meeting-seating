# 会议桌牌自动排序工具

面向公司行政人员的**会议桌牌自动排序工具**。输入客方、主方人数，工具自动按座次礼仪规则生成可编辑的 SVG 桌牌摆放平面图，支持 PNG 下载拖入 PPT/Word 编辑使用。

## 在线地址

🔗 [https://iann-test.github.io/meeting-seating/](https://iann-test.github.io/meeting-seating/)

## 功能特性

- **智能座次排列**：基于"以左为尊、居中为尊、面门为尊、主客分明、1号对坐"六大规则，自动计算座次
- **SVG 平面图预览**：实时渲染标准长方形会议室（18座布局），含电子白板、门位置标注
- **PNG 下载**：一键导出高清 PNG（2x 分辨率），可直接拖入 PPT/Word
- **自定义微调**：支持拖拽交换座位、双击编辑姓名，灵活适配实际需求
- **移动端适配**：导航栏、输入卡片、预览区均适配手机竖屏布局

## 会议室布局

```
       短边2座（占位）
  ———————————————————
  |                 |      门1
  | 长边A 8座（客方面门）  |
  |                 | 长
  |                 | 边
  |   会  议  桌    | B
  |                 | 8
  | 长边B 8座（主方门侧）  | 座
  |                 |      门2
   ——————————————————
      电子白板（无座）
```

## 座次规则

| # | 规则 | 说明 |
|---|------|------|
| 1 | 以左为尊 | 2号在1号左手边，3号在右手边，依次左右交替 |
| 2 | 居中为尊 | 1号永远居中；偶数时1号、2号同时居中 |
| 3 | 面门为尊 | 客方面对门（长边A），主方背对门（长边B） |
| 4 | 主客分明 | 客方坐长边A，主方坐长边B，互不混排 |
| 5 | 1号对坐 | 无论人数奇偶，双方1号始终正面对坐 |
| 6 | 列席位处理 | 双方均满员时，主方8号以对比色标识为列席 |

## 技术栈

| 层面 | 技术 |
|------|------|
| 页面 | 纯静态单文件 HTML5 |
| 样式 | CSS 变量 + OKLCH 色彩空间 |
| 渲染 | 原生 SVG（JS 动态生成座位层） |
| 图片导出 | Canvas API（SVG → PNG，2x 分辨率） |
| 布局 | CSS Grid（桌面双栏）/ Flexbox（移动端单列） |
| 部署 | GitHub Pages |
| 依赖 | **零外部依赖**，双击 `index.html` 即可运行 |

## 本地运行

```bash
# 方式一：直接打开
双击 index.html

# 方式二：本地静态服务器（可选）
python -m http.server 8000
# 浏览器打开 http://localhost:8000
```

无需 `npm install`，无需任何构建步骤。
